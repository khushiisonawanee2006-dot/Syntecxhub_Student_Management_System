# Syntecxhub_Student_Management_System
Student Management System

This is a simple Python project made to manage student records. Users can add, update, delete, and view student details like name, ID, and grade. The data is stored in a JSON file so records are saved permanently.

Features

- Add Student
- Update Student
- Delete Student
- View All Students
- JSON File Storage
- Unique ID Validation

Technologies Used

- Python
- JSON
- File Handling

This project helped me practice Python basics, functions, loops, dictionaries, and file handling.
